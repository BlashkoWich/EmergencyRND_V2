# EmergencyRND V2 — Project Specs

## IMPORTANT: No Testing
**НИКОГДА не тестировать при разработке — ни в браузере, ни иным способом.** Вся необходимая информация для реализации фичей содержится в spec-файлах. Любое тестирование запрещено — пиши код, опираясь исключительно на спецификацию.

## IMPORTANT: Локализация
**Все тексты в игре ОБЯЗАНЫ быть на всех доступных языках (ru, en).** Никогда не хардкодить строки — использовать `Game.Lang.t('key')`. Подробнее — см. [specs-lang.md](specs/specs-lang.md) и [specs-base.md](specs/specs-base.md).

## Spec Files

Спецификация разделена на файлы в папке `specs/`:

- **[specs-base.md](specs/specs-base.md)** — техническая база: стек, архитектура, файловая структура, рендерер, animation loop, DOM элементы, z-index, API модулей
- **[specs-scene.md](specs/specs-scene.md)** — сцена: здание, мебель, улица, освещение, коллизии, 3D модели (пациенты, расходники, стеллажи, индикаторы), координаты
- **[specs-player.md](specs/specs-player.md)** — игрок: управление (WASD, мышь, клавиши), камера, UI (прицел, подсказки, overlay), инвентарь, магазин, взаимодействие со стеллажами и расходниками
- **[specs-patient.md](specs/specs-patient.md)** — пациенты: данные, состояния, горизонтальная очередь, спавн, попап, тяжесть, HP-бар и потеря, мульти-препаратная система, авто-маршрутизация после диагностики, движение
- **[specs-treatment.md](specs/specs-treatment.md)** — лечение: мульти-препаратная механика (через медсестру), индикаторы над кроватью, авто-выписка на кассу после последнего препарата, анимации, частицы
- **[specs-cashier.md](specs/specs-cashier.md)** — касса: самообслуживание, баланс игрока, снятие денег (hold E), уход пациентов
- **[specs-diagnostics.md](specs/specs-diagnostics.md)** — диагностика: только сотрудник-диагност (мини-игры игрока удалены), длительность 30–45с, авто-маршрутизация результата
- **[specs-staff.md](specs/specs-staff.md)** — сотрудники: пре-хайр диагноста и медсестры на старте, зарплата отключена, триаж по min(HP), длительность работы 30–45с
- **[specs-shift.md](specs/specs-shift.md)** — смены: 5-минутные дни, авто-спавн, попап конца дня без зарплаты
- **[specs-ads.md](specs/specs-ads.md)** — реклама за деньги: попап предложения, фиктивный 20-секундный ролик, награда $200, заморозка игры
- **[specs-lang.md](specs/specs-lang.md)** — локализация: система переводов (ru/en), Game.Lang API, выбор языка, правила добавления текстов

Note: `specs-tutorial.md` — устарел (туториал и Level 1 удалены).
