# EmergencyRND V2 — Реклама за деньги (Game.Ads)

## Концепция
Когда у игрока недостаточно средств для покупки в магазине, вместо уведомления "Недостаточно средств!" открывается попап с предложением посмотреть рекламу за $200. Реклама фиктивная — анимированный CSS-ролик ~20 секунд. После просмотра деньги зачисляются на баланс.

## Попап предложения (`#ad-offer-popup`)
- z-index: 22 (выше магазина z:20, ниже рекламного оверлея z:50)
- Появляется при попытке купить товар без достаточного баланса
- Стиль: тёмный попап (`#1a2a3a`), центрирован, скруглён (14px)
- Заголовок: `Game.Lang.t('ad.offer.title')` (красный, `#ff6b6b`)
- Текст: `Game.Lang.t('ad.offer.text')` (с зелёной суммой `#4ade80`)
- Две кнопки:
  - `Game.Lang.t('ad.watch')` — зелёная (`#2a8a5a`), запускает рекламу
  - `Game.Lang.t('ad.decline')` — серая (rgba), закрывает попап

## Рекламный оверлей (`#ad-overlay`)
- z-index: 50 (поверх всего)
- Fullscreen, чёрный фон (`position: fixed; inset: 0`)
- Игра полностью заморожена (`Game.Ads.isFrozen() === true`) — animation loop рендерит сцену, но не вызывает `update()` модулей

### Контент рекламы (`#ad-content`)
- Область 700x400px, скруглена, градиентный фон (`#1a1a2e → #16213e`)
- 4 слайда по 5 секунд каждый (CSS `@keyframes ad-slide-show`):
  1. **Слайд 1**: Логотип "MEGAMART" (оранжевый, `#ff6b35`, с `ad-logo-glow` анимацией пульсации свечения) + "MEGA SALE!" + подтекст
  2. **Слайд 2**: Эмодзи-продукт (bounce-анимация `ad-bounce`, 1.5s, infinite) + "Premium Quality" + подтекст
  3. **Слайд 3**: "-50%" (золотой `#ffd700`, `ad-pulse-scale` пульсация масштаба 1→1.1) + "Limited Time Offer!"
  4. **Слайд 4**: "BUY NOW!" (красный `#ff4444`, мигающая рамка `ad-cta-blink`) + мелкий дисклеймер

### Слайд-анимация (`@keyframes ad-slide-show`)
- 0%: `opacity: 0, scale(0.8), translateY(30px)` — вход снизу
- 10%: `opacity: 1, scale(1), translateY(0)` — показ
- 85%: видимый
- 100%: `opacity: 0, scale(1.05), translateY(-20px)` — уход вверх
- Delay: слайд 1 = 0s, слайд 2 = 5s, слайд 3 = 10s, слайд 4 = 15s

### Прогресс-бар (`#ad-progress-bar` + `#ad-progress-fill`)
- Внизу оверлея, высота 6px, зелёная заливка (`#4ade80`)
- Ширина заполняется от 0% до 100% за 20 секунд
- Обновляется JS-интервалом каждую секунду, `transition: width 1s linear`

### Таймер (`#ad-timer`)
- Внизу справа, показывает `Game.Lang.t('ad.timer', [remaining])` обратный отсчёт
- При завершении: `Game.Lang.t('ad.done')`

### Кнопка закрытия (`#ad-close-btn`)
- Вверху справа, круглая (36x36px), крестик `✕`
- Полупрозрачный фон (`rgba(255,255,255,0.15)`), z-index: 52
- При нажатии показывает диалог подтверждения (не закрывает рекламу сразу)

### Диалог подтверждения закрытия (`#ad-confirm-dialog`)
- Абсолютно позиционирован поверх рекламы (z-index: 53)
- Полупрозрачный чёрный фон (`rgba(0,0,0,0.85)`)
- Текст: `Game.Lang.t('ad.confirm')`
- `Game.Lang.t('ad.confirmYes')` — красная кнопка (`#8b3333`), отменяет рекламу без награды
- `Game.Lang.t('ad.confirmNo')` — зелёная кнопка (`#2a8a5a`), скрывает диалог, реклама продолжается

## Анимация награды (`#ad-reward-animation`)
- z-index: 50, fullscreen, тёмный фон (`rgba(0,0,0,0.85)`)
- "+$200" — крупный зелёный текст (5rem, `#4ade80`), анимация `ad-reward-pop`:
  - 0%: `scale(0), opacity(0)`
  - 60%: `scale(1.2), opacity(1)` — пружинящий эффект (`cubic-bezier(0.34, 1.56, 0.64, 1)`)
  - 100%: `scale(1), opacity(1)`
- `Game.Lang.t('ad.reward')` — подтекст, появляется с задержкой 0.4s (`ad-reward-fade`)
- Длительность: 2.5 секунды, затем автоматически закрывается
- Деньги зачисляются через `Game.Cashier.earn(200)` в момент показа анимации

## Заморозка игры
- Флаг `frozen` в модуле Ads
- В animation loop (`index.html`): `if (Game.Ads && Game.Ads.isFrozen()) { render; return; }` — рендер сцены продолжается, но все `update()` модулей пропускаются
- Заморозка включается при старте рекламы, выключается при завершении/отмене + после reward-анимации

## Интеграция с магазином
- В `shop.js` все 4 проверки `balance < price` (расходники, инструменты, мебель, апгрейды) вызывают `Game.Ads.show()` вместо `Game.Inventory.showNotification('Недостаточно средств!')`
- Попап предложения открывается поверх магазина (z-index 22 > 20)
- При закрытии предложения ("Нет, спасибо") игрок возвращается в магазин

## Интеграция с controls
- В `controls.js` unlock handler: `if (Game.Ads && Game.Ads.isActive()) return;` — предотвращает показ паузы при активной рекламе

## Internal State
```js
AD_REWARD = 200           // сумма награды
AD_DURATION = 20          // длительность рекламы (секунд)
frozen = false            // заморозка игры
active = false            // любой UI рекламы показан (offer/overlay/reward)
adElapsed = 0             // прошло секунд рекламы
adInterval = null         // setInterval ID для таймера
```

## Public API (`Game.Ads`)
- `setup()` — кеширование DOM-ссылок, привязка обработчиков
- `show()` — открыть попап предложения
- `isActive()` → boolean — показан ли любой UI рекламы
- `isFrozen()` → boolean — заморожена ли игра (для animation loop)
